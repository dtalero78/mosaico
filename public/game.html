🔌 Hook
Capa cliente. Hace fetch al API y maneja cache con React Query. Hay 8 hooks.
Ej: useStudentProfile(id) llama a /api/postgres/students/{id}
 → 🚪 API Route
Adaptador HTTP delgado. Recibe el request, llama al servicio, devuelve respuesta. ~95 rutas.
Ej: postgres/students/[id]/route.ts usa handlerWithAuth()
 → ⚙ Service
Lógica de negocio. Sabe las "recetas": buscar en ACADEMICA, si no, buscar en PEOPLE. 22 servicios.
Ej: studentService.getProfile() combina datos de 2 tablas
 → 📦 Repository
Acceso a datos. SQL parametrizado ($1, $2...). Un repo por tabla. 21 repos.
Ej: peopleRepository.findById($1) → SELECT * FROM "PEOPLE"
 → 🗃 PostgreSQL
Base de datos PostgreSQL en Digital Ocean. Tablas: PEOPLE, ACADEMICA, CALENDARIO, BOOKINGS, etc.
Ej: 10+ tablas con datos de estudiantes, eventos, contratos
